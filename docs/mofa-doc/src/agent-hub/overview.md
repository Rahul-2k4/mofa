# Agent Hub Overview

The Agent Hub is the discovery layer for reusable MoFA nodes and agent building blocks.

## Purpose

The hub gives contributors and application developers one place to:

- discover available nodes
- inspect metadata and capabilities
- copy ready-to-use snippets into MoFA Studio flows

## Source of Truth

Hub registry data is sourced from:

- `https://raw.githubusercontent.com/mofa-org/mofa-node-hub/main/registry.json`

When hub metadata changes, update documentation to reflect:

- required fields
- recommended fields
- compatibility notes

## Ownership

- Registry content ownership: `mofa-org/mofa-node-hub`
- Integration/documentation ownership: `mofa-org/mofa` docs maintainers
