# Agent Hub 目录

本页定义了集中式文档所使用的目录模型和用户体验行为。

## 目录字段

每条目录条目至少应包含以下字段：

- `name`（名称）
- `description`（描述）
- `category`（类别或等效分组）
- `version`（版本）
- `source`（仓库或包链接）

推荐字段：

- `tags`（标签）
- `inputs` 和 `outputs` 摘要
- 兼容性约束（运行时/提供商/版本）
- 示例用法片段

## 搜索与筛选

目录体验应支持：

- 按 `name`、`description` 和 `tags` 进行全文搜索
- 按类别筛选
- 在有兼容性标签时按兼容性标签筛选

## 缓存指南

对注册表请求使用客户端缓存，TTL 建议设置为 5–10 分钟，以减少重复请求并提升页面响应速度。

## 故障处理

当 Hub 数据源不可用时：

- 优先展示上次已知的缓存快照
- 否则显示明确的不可用状态
- 提供直接链接至上游注册表 URL，便于调试
