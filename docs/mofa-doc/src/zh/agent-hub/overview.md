# Agent Hub 概览

Agent Hub 是可复用 MoFA 节点和智能体构建块的发现层。

## 目的

Hub 为贡献者和应用开发者提供统一的入口，用于：

- 发现可用节点
- 查看元数据和能力描述
- 将即用型片段复制到 MoFA Studio 数据流中

## 数据来源

Hub 注册表数据来源于：

- `https://raw.githubusercontent.com/mofa-org/mofa-node-hub/main/registry.json`

当 Hub 元数据发生变更时，请同步更新文档以反映：

- 必填字段
- 推荐字段
- 兼容性说明

## 维护归属

- 注册表内容维护：`mofa-org/mofa-node-hub`
- 集成与文档维护：`mofa-org/mofa` 文档维护者
