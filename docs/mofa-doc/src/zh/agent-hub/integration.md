# Agent Hub 集成

本节介绍 Hub 条目如何与 MoFA 文档和开发者工作流进行关联。

## 集成目标

Hub 元数据应能清晰映射到：

- `docs/mofa-doc/src/` 中的文档页面
- `examples/` 中的可运行示例
- Studio 导向的片段指南

## 文档映射

添加新节点或智能体类型时：

1. 添加或更新上游注册表条目
2. 添加或更新相关文档页面
3. 添加至少一个参考示例（若行为较复杂时）

## 更新流程

遇到过时或错误的条目时，请按以下路径处理：

1. 在 `mofa-org/mofa-node-hub` 提交 issue/PR 修复注册表
2. 若文档映射也需变更，在 `mofa-org/mofa` 提交文档更新 PR
3. 关联两个 PR 以便追溯

## 验证清单

- 条目元数据可正确解析
- 关联文档页面存在
- 示例/片段引用未失效
- 必要时提供兼容性说明
